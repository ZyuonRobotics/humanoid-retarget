# Web 前端使用说明

本文档详细介绍 Humanoid Retargeting Web 前端的使用方法。

---

## 1. 重定向器

### 1.1 重定向流程

1. 确保待重定向的动捕文件的人体配置已经设置（在播放器->人体动作->人体配置中设置）
2. 在"重定向器"页面选择机器人、动捕文件格式(bvh,smpl)和动捕文件
3. 加载或配置重定向参数
4. 点击"重定向"按钮
5. 等待处理完成
6. 完成后可以在可视化界面查看结果


### 1.2 重定向配置说明

#### 1.2.1 机器人设置

**选择机器人模型：**
1. 从下拉菜单中选择目标机器人模型
2. 系统支持的机器人模型通过 `hurodes` 库提供
3. 常见机器人模型包括：
   - DumBot13-21dof
   - unitree-g1-29dof
   - 其他自定义机器人模型

**机器人相关参数：**
- 机器人的髋关节、膝关节、脚部等关键部位由hurodes定义(见HURODES_ASSETS_PATH/robots/robot_name/meta.yaml)
- Base Link X\Y 偏移表示机器人和人体的相对位置，通常均为0
- Base Link 旋转为动捕文件坐标系和机器人坐标系的关系，通常和文件格式绑定，一般不需要修改。
- 系统会自动计算机器人的腿长等物理参数

#### 1.2.2 人体参数

人体参数需要在播放器的"人体配置"中预先设置（详见 2.1.1 人体配置）。人体参数用于计算人体与机器人的比例关系。
其缩放方法为：计算机器人和人体髋到脚的长度，进行等比例缩放。

**身体比例调整：**
- **extra_body_ratio**: 全局身体比例缩放 [x, y, z]（默认 [1.0, 1.0, 1.0]）
  - 用于整体调整人体模型的尺寸以匹配机器人
- **relative_body_ratio_dict**: 相对身体比例字典
  - 针对特定身体部位的局部比例调整
  - 格式：`{"body_name": [x, y, z]}`，代表每个方向缩放的比例

**身体旋转调整：**
- **body_rotate_dict**: 身体部位旋转字典
  - 针对特定身体部位的旋转调整
  - 格式：`{"body_name": [roll, pitch, yaw]}`


#### 1.2.3 重定向参数

**跟踪器配置 (tracker_dict)：**
- 定义人体关键点与机器人关键点的对应关系
- 每个跟踪器组包含：
  - **human**: 人体关键点名称列表
  - **robot**: 对应的机器人关键点名称列表
  - **position_cost**: 位置匹配权重（控制位置跟踪的重要性）
  - **orientation_cost**: 方向匹配权重（控制姿态跟踪的重要性）

**优化参数：**
- **damping_cost**: 阻尼代价系数（默认 5.0）
  - 用于平滑运动，防止抖动
  - 值越大，运动越平滑但可能损失精度

#### 1.2.4 保存和加载配置

1. **保存配置**：
   - 配置完参数后，点击"保存配置"按钮
   - 输入配置名称（例如：`default_config`）
   - 配置文件保存在 `$RETARGETING_PATH/configs/{robot_name}/{config_name}/`
   - **注意每次重定向时要对配置进行保存，否则修改不会生效**

2. **加载配置**：
   - 点击"加载配置"按钮
   - 从列表中选择已保存的配置
   - 系统会自动填充所有参数

## 2. 播放器

### 2.1 人体动作播放

#### 2.1.1 动作格式

**支持的人体模型格式：**
- **SMPL 格式** (`.npz` 文件)：
  - 必需字段：`trans`, `poses`, `betas`, `mocap_framerate`, `gender`
  - SMPL 是参数化人体模型，通过形状参数 (betas) 和姿态参数 (poses) 表示人体
  - 支持不同性别模型：male, female, neutral
  
- **BVH 格式** (`.bvh` 文件)：
  - 标准 BVH 格式规范
  - 包含骨骼层级结构和运动数据
  - 广泛用于动作捕捉和动画制作

#### 2.1.2 人体配置
人体配置是针对每个动作捕捉文件的特定配置，是重定向必须的配置。该配置负责计算人体与机器人的比例关系、调整人体高度、调整人体关节偏移。

**必需参数：**
- **hip_names**: 髋关节名称列表（左右髋关节）
- **hip_offset**: 髋关节偏移量
- **foot_names**: 脚部名称列表（左右脚）
- **foot_offset**: 脚部偏移量
- **height_adjustment**: 高度调整值，用于地面对齐(点击自动计算即可)
- **height_adjustment_method**: 高度调整方法（`plane_fit` 或 `offset`）

**高度调整说明**
由于动捕设备的偏差，导致平面是倾斜的。我们采用了平面拟合或偏移的方法处理。平面拟合（`plane_fit`）的参数a、b、c通过计算$height = ax + by + c$确认一块平面，对人体高度进行修正。

## 3. 工具箱

### 3.1 人体动作切割、机器人动作切割
